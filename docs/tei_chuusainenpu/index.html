    TEI 抽斎年譜ビューワ #

    TEI 抽斎年譜ビューワ トップページ

    「TEI 抽斎年譜ビューワ」について #

    東京大学総合図書館所蔵『抽斎年譜』は、森鷗外が史伝小説『渋江抽斎』の執筆に際して抽斎の七男・渋江保に制作を依頼した年譜資料です。年譜の本文および欄外には、作成者である渋江保の多様な書き込みに加え、鷗外自身による書入れが見られます。 年譜の読解にあたっては、注の位置・インデント・朱筆/墨筆といったテキストの形態的な情報、および書入れを行った人物に関する情報を確認する必要があります。そこで、年譜の本文テキストをTEI P5ガイドラインに準拠したXMLタグを用いて記述し、あわせて、記述内容を反映した『抽斎年譜』の可視化サイトを構築しました。 また、年譜には、鷗外が『渋江抽斎』を執筆する際に典拠としたと思われる箇所が散見されます。文献研究の一助となることを目指して、両資料における対応箇所についてもTEI/XMLで記述し、ビューワ上で容易に確認できる機能を実装しました。

    機能 #

    1. 本文と対応する画像の表示 『抽斎年譜』の本文を表示するテキストビューワと、画像を表示する画像ビューワを並べて表示しています。テキストビューワには、半丁ごとにIIIFのアイコンが設置されています。アイコンをクリックすると、左の画像ビューワ（Mirador）が遷移し、対応する画像コマが表示されます。

    2. 注・書入れの特徴に応じた表示 注や書入れの形態的な特徴（文字色やインデント、出現位置等）に応じて、テキストビューワ上での注・書入れの表示を調整しています。また、『抽斎年譜』の資料的特徴である森鷗外による書入れ部分は、ハイライト表示で強調しています。

    3. 人物情報の表示 「人物情報」タブでは、『抽斎年譜』に登場する抽斎の係累や友人の人物情報をカード表示しています。「本文中の参照箇所を表示」ボタンをクリックすると、本文中の出現箇所がハイライトで表示され、各人物について年譜中の具体的な記述を確認することができます。

    4. タイムライン表示 『抽斎年譜』は編年体の構成をとりますが、記載されている出来事は必ずしも時系列順に並んでいる訳ではありません。そこで、「タイムライン」タブでは、TEI/XML上で要素の@when に記述した西暦の情報をもとに、各出来事を時系列に並べて表示しています。

    5. 『渋江抽斎』との対応箇所の表示 先行研究において、史伝小説『渋江抽斎』は『抽斎年譜』に大きく依拠するとされています。そこで、「青空文庫との比較」タブでは、『抽斎年譜』の記載が『渋江抽斎』に描かれた内容と対応している箇所にリンクボタンを設置しました。リンクボタンをクリックすると、画面下の『渋江抽斎』テキストビューワが遷移し、該当箇所が表示されます。

    TEI/XML公開 #

    テキストビューワで表示している『抽斎年譜』および『渋江抽斎』のTEI/XMLを、クリエイティブ・コモンズ 表示 4.0 国際（CC BY 4.0）のもとにGitHub上で公開します。

    • 若手図書館員DH勉強会 GitHubリポジトリ

    TEIマークアップの方針 #

    • 『抽斎年譜』のTEI/XMLは、TEI P5ガイドラインに準拠して作成しました。割注やルビのマークアップにあたっては、TEI 協会東アジア/日本語分科会の日本語古典籍TEI 本文データ作成要領を参照しています。エンコーディングの方針については、TEI/XMLのTEIヘッダーおよび「抽斎年譜マークアップ方針」をご参照ください。
    • 『渋江抽斎』のTEI/XMLは、青空文庫で公開されているHTMLファイルをXMLに自動変換し、『抽斎年譜』との対応箇所にタグを挿入して作成しました。
    • 『渋江抽斎』と『抽斎年譜』の対応箇所として、『日本近代文学大系12 森鷗外集Ⅱ』所収『渋江抽斎』の補注に掲出されている箇所をマークアップしました。なお、頭注にも『抽斎年譜』に関する記述がありますが、現時点ではTEI/XMLに反映していません。今後の更新での反映を予定しています。
